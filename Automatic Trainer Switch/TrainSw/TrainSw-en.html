Automatic Trainer Switch

Description

The application is useful for users who take advantage of the Teacher/Student mode. It allows you to play an audio file to indicate the model control handover between the teacher and student. Switching controls to the teacher is done automatically as soon as he/she moves any transmitter gimbal. The teacher thus gains immediate control of the model.

Usage

  1. In the menu, choose Applications - User Applications. Press the F(3) key to open the application browser. Select the "TrainSw" file here.

  2. Open the application configuration. Here you can choose the appropriate audio files and a switch:

    • Audio Teacher - the audio file that will be played when the teacher handles the model.
    • Audio Student - the audio file to be played when the student gets the control.
    • Switch - a switch by which the teacher passes control to the student.
  3. In the Advanced Properties - Wireless Modes/Trainer switch the transmitter to the Teacher mode and select a virtual input "T/S" in the Apps section F(5) as a switch between teacher/student.

You can download the source code from the official JETImodel Lua-Apps GitHub repository.
