Sensor Chart

Description

  • Allows you to view a single telemetry variable into a chart on the Desktop.

    Usage

    1. In the menu, choose Applications - User Applications. Press the F(3) key to open the application browser. Select the "Sensors" file here.

    2. Open the application configuration. Select the sensor of interest and its possible range of minimum and maximum values (currently the range is not automatic).

    3. In the menu, select Timers/Sensors - Displayed Telemetry . Create a new item here and select "Telemetry Chart". The application will have its own desktop widget.

    You can download the source code from the official JETImodel Lua-Apps GitHub repository.
